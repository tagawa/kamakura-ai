**状況**: AIの間違いを指摘すると「以後気をつけます」と返すが、次の会話では同じ失敗を繰り返す。

**やったこと**: 間違いのたびに「原因と、次回どうすれば防げるかを短く書いて」と頼み、その文章を次回も読み込まれる場所(カスタム指示、プロジェクトのメモ、指示ファイルなど)に自分で貼って保存する。

**結果・ポイント**: AIの「覚えます」という約束は会話が終われば消える。教訓は文章にして、次のセッションが実際に読める場所に置いて初めて効く。溜まったメモは自分専用の取扱説明書になる。

<!--en-->
**Situation**: When you point out an AI's mistake, it promises to be careful, then repeats the mistake in the next conversation.

**What to do**: Each time something goes wrong, ask: "Write a short note on what caused this and how to prevent it next time," then save that note yourself somewhere the AI reads in future sessions (custom instructions, project notes, an instructions file).

**Result / key point**: A promise to remember evaporates when the chat ends. Lessons only stick once written where the next session can actually read them. Over time the notes become your personal manual.
